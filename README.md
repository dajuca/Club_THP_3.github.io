# Testing on Rails

Des tests en série, pétages de câble absolus au codage...

## Prérequis

Ruby Version : 2.5.1

Rails Version : 5.2.0

pg-1.1.3.gem

## Déploiement en local

``git clone``

```cd THP_Testing_Rails```

```rails db:create```

```rails db:migrate```

```rails db:seed```

```rails test```
